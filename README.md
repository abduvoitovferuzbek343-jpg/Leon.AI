# 🤖 Leon.AI

Leon.AI is a simple but intelligent AI-inspired web project designed to simulate a smart digital assistant experience. It focuses on clean design, fast interaction, and a futuristic AI vibe.

## 🚀 About the Project

Leon.AI is a personal project created to explore AI assistant-style interfaces. The goal is to build a minimal but powerful looking AI system that feels like a real digital companion.

This website includes a modern UI and represents the idea of an intelligent assistant that can grow into a more advanced system in the future.

## 💡 Features

- Clean and simple AI-style interface  
- Fast and lightweight design  
- Futuristic assistant concept  
- Beginner-friendly web structure
